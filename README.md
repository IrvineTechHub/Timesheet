# 📑 Timesheet 📑
온라인 타임시트 사이트

## ☀️ 프로젝트 소개
손글씨로 적은 타임시트를 사진으로 찍어서 직원들의 checkin 을 관리하는 사이트
직원들의 근무 시간과 급여를 확인할 수 있다.
erpnext를 통해 체계적으로 관리 가능

## 🔨 개발 환경
- Frontend : react
- Backend  : flask
- Database : mysql
